# 보험한약 처방 도우미

건강보험 적용 한약제제 56종 중에서, 환자의 증상에 맞는 처방을 추천하는 웹 기반 문진 도구입니다.

> **주의**: 이 도구는 참고용이며, 실제 진료를 대체하지 않습니다. 정확한 진단과 처방은 반드시 한의사와 상담하세요.

---

## 작동 원리

사용자가 6단계 문진에 답하면, 56종 처방 각각에 대해 점수를 계산하고 **상위 3개**를 추천합니다.

```
[STEP 1] 기본 정보 (성별, 연령, 임신, 체형)
    ↓
[STEP 2] 주증상 계통 선택 (8개 중 1개)
    ↓
[STEP 3] 세부 증상 (계통별 조건부 분기)
    ↓
[STEP 4] 체질 문진 (한열, 허실, 대변, 식욕, 수면)
    ↓
[STEP 5] 동반 증상 (복수 선택)
    ↓
[STEP 6] 발병 경과 (시기, 악화 요인)
    ↓
[점수 계산] → 상위 3개 추천
```

---

## 각 단계별 상세

### STEP 1. 기본 정보

| 질문 | 선택지 | 알고리즘 영향 |
|------|--------|--------------|
| 성별 | 남성 / 여성 | 여성 → 임신 여부 질문 활성화 |
| 연령대 | 소아 / 청소년 / 성인 / 고령 | 소아 → 시호청간탕 +10점 |
| 임신 여부 | 예 / 아니요 | 임신 → 임산부 처방 +20점, 발한·사하 처방 -15점 |
| 체형 | 마른편 / 보통 / 비만편 | 참고용 |

**임신 시 감점되는 처방** (발한·사하 작용이 강한 처방):
갈근탕, 갈근해기탕, 대청룡탕, 구미강활탕, 소청룡탕, 대황목단피탕, 도인승기탕, 조위승기탕, 대시호탕, 오적산

---

### STEP 2. 주증상 계통

8개 계통 중 **가장 불편한 1개**를 선택합니다.

| 계통 | 설명 | 대표 처방 |
|------|------|----------|
| 호흡기 | 감기, 기침, 코막힘, 인후통 | 갈근탕, 소청룡탕, 인삼패독산 |
| 소화기 | 복통, 소화불량, 구역, 설사, 변비 | 반하사심탕, 평위산, 이중탕 |
| 근골격계 | 요통, 관절통, 근육통 | 오적산, 갈근탕, 구미강활탕 |
| 신경/정신 | 두통, 어지러움, 불면, 불안 | 반하백출천마탕, 복령보심탕 |
| 비뇨생식기 | 소변이상, 월경불순 | 오림산 |
| 피부 | 발진, 가려움, 종기 | 당귀연교음, 황련해독탕 |
| 산부인과 | 임신 관련 증상 | 궁소산, 안태음 |
| 전신 | 만성피로, 기력저하, 발열 | 보중익기탕, 팔물탕 |

선택한 계통이 처방의 `systems`에 포함되면 **+10점**.

---

### STEP 3. 세부 증상 (조건부 분기)

STEP 2의 선택에 따라 **서로 다른 질문**이 표시됩니다.

#### 호흡기 선택 시
```
세부 증상 선택
├── 갑자기 시작된 감기 → 감기 양상 추가 질문
│   ├── 오한 심하고 땀 안 남    → 갈근탕, 갈근해기탕 (+6)
│   ├── 오한과 함께 땀이 남      → 삼소음, 소청룡탕 (+6)
│   ├── 열감 주되고 인후통 동반   → 연교패독산 (+6)
│   └── 감기 오래되어 낫지 않음   → 소시호탕, 시호계지탕 (+6)
├── 기침·가래 → 이진탕, 행소탕, 시경반하탕
├── 코막힘·콧물·비염 → 소청룡탕(+4 MFDS보너스), 형개연교탕
├── 인후통·편도부종 → 연교패독산, 시호청간탕
└── 마른기침 → 자음강화탕, 생맥산
```

#### 소화기 선택 시
```
세부 증상 선택
├── 식욕없고 더부룩(소화불량) → 평위산, 향사평위산, 삼출건비탕
├── 구역·구토 → 반하사심탕, 불환금정기산, 이진탕
├── 설사 → 이중탕, 승양보위탕
├── 변비 → 조위승기탕, 대시호탕, 대황목단피탕
├── 복통 → 대화중음, 이중탕
├── 속쓰림·위산역류 → 반하사심탕(+6 MFDS보너스)
├── 인후이물감(매핵기) → 반하후박탕
└── 치통·구내염 → 청위산, 황련해독탕

+ 위장 상태 추가 질문
├── 평소 소화력 약함 → 식욕부진 처방 +5
├── 스트레스성 → 심하비 처방 +5
├── 과식·음주 후 → 과식 처방 +5
└── 만성 반복 → 만성 처방 +3
```

#### 근골격계 선택 시
```
세부 증상 + 통증 양상
├── 급성 염좌
├── 만성 요통·관절통
├── 목·어깨 뻣뻣함 → 갈근탕 (+6 +4 MFDS보너스)
└── 전신 근육통·몸살 → 오적산, 구미강활탕 (+5)

통증 양상
├── 추위에 악화 → 한증 처방 +5, 오적산 추가 +5
├── 찌르는 고정통 → 어혈 처방 +6
└── 부종+열감 → 열증 처방 +5
```

#### 신경/정신 선택 시
```
├── 두통 → 청상견통탕, 반하백출천마탕
├── 어지러움 → 반하백출천마탕
├── 불면증 → 불면 세부 질문
│   ├── 잠들기 어려움(입면장애) → 가미소요산, 삼황사심탕
│   └── 자주 깸/꿈 많음 → 복령보심탕
├── 불안·초조 → 복령보심탕, 가미소요산
└── 우울감 → 가미소요산
```

#### 전신 증상 선택 시
```
├── 만성 피로 → 보중익기탕, 팔물탕
├── 식은땀 → 자음강화탕, 당귀육황탕
├── 여름 더위 → 청서익기탕
├── 미열 지속 → 자음강화탕
└── 갱년기(안면홍조, 감정기복) → 가미소요산 (+8 MFDS보너스)
```

세부 증상이 처방의 `symptoms`에 포함되면 **+8점**.

---

### STEP 4. 체질 문진 (한열·허실 변증)

이 단계가 처방 분류의 **핵심 감별 기준**입니다.

#### 한열(寒熱) 감별

| 선택 | 의미 | 해당 처방 방향 |
|------|------|---------------|
| 추위를 많이 탐 | 한증(寒證) | 이중탕, 오적산, 대화중음 |
| 열이 많음 | 열증(熱證) | 황련해독탕, 삼황사심탕, 조위승기탕 |
| 보통 | 중간 | 제한 없음 |

- 한열 일치: **+5점**
- 한열 불일치: **-3점** (반대 성질의 처방은 감점)

#### 허실(虛實) 감별

| 선택 | 의미 | 해당 처방 방향 |
|------|------|---------------|
| 기운 없고 쉽게 피로 | 허증(虛證) → 보(補)하는 처방 | 보중익기탕, 팔물탕, 삼소음 |
| 기력 괜찮으나 증상 불편 | 실증(實證) → 사(瀉)하는 처방 | 대시호탕, 조위승기탕, 대청룡탕 |
| 최근 갑자기 악화 | 중립 | 제한 없음 |

- 허실 일치: **+4점**
- 허실 불일치: **-2점**

#### 보조 체질 질문

| 질문 | 점수 영향 |
|------|----------|
| 대변: 설사 → 설사처방 +3, 변비처방 -2 | 변비 → 변비처방 +3, 설사처방 -2 | 교대 → 반하사심탕 +4 |
| 식욕: 없고 물 안마심 → 한허증 +3 | 식욕 왕성+찬물 → 열증 +3 | 먹으면 불편 → 체증 +2 |
| 수면: 입면장애 → insomnia_difficulty +4 | 잘 깸 → insomnia_waking +4 |

---

### STEP 5. 동반 증상 (복수 선택 가능)

| 동반 증상 | 관련 처방 예시 |
|-----------|---------------|
| 가래가 많다 | 소청룡탕, 이진탕, 오적산 |
| 가슴 두근거림 | 복령보심탕 |
| 입이 마르거나 쓰다 | 삼황사심탕, 청위산, 소시호탕 |
| 눈 충혈 | 삼황사심탕, 청상견통탕 |
| 식은땀 | 자음강화탕, 보중익기탕, 가미소요산 |
| 손발이 차다 | 이중탕, 오적산 |
| 소변 진함 | 오림산 |
| 피부 발진 | 형개연교탕, 황련해독탕 |
| 월경 관련 문제 | 도인승기탕, 가미소요산 |

동반 증상 1개 일치당 **+2점**.

---

### STEP 6. 발병 경과

| 질문 | 점수 |
|------|------|
| 발병 시기 일치 (급성/아급성/만성) | **+3점** |
| 악화 요인 일치 (추위/더위/스트레스/과식) | **+4점** |

---

## 점수 계산 공식

```
총점 = 기본 보정값 (score_boost)
     + 임산부/소아 보정
     + 주증상 계통 일치 (+10)
     + 세부 증상 일치 (+8)
     + 감기/통증/소화 양상 보너스 (+3~6)
     + 한열 변증 (+5 / -3)
     + 허실 변증 (+4 / -2)
     + 대변/식욕/수면 보정 (+2~4)
     + 동반 증상 (개당 +2)
     + 발병 시기 (+3)
     + 악화 요인 (+4)
     + MFDS 특별 보너스
```

### MFDS(식약처) 허가사항 기반 특별 보너스

| 처방 | 조건 | 보너스 | 근거 |
|------|------|--------|------|
| 오적산 | 근골격+추위악화 통증 | +5 | "냉증에 의한 복통, 요통" |
| 갈근탕 | 목/어깨 뻣뻣함 | +4 | "어깨결림, 근육통, 상지신경통" |
| 가미소요산 | 갱년기 증상 | +8 | "갱년기장애, 안면홍조" |
| 반하사심탕 | 속쓰림 | +6 | "트림, 속쓰림, 위산역류" |
| 소청룡탕 | 비염 증상 | +4 | "비염, 묽은 가래가 많은 기침" |

## 생산중단 처방 (10종)

아래 처방들은 현재 제약사에서 생산하지 않아 `score_boost: -5`로 설정되어 있습니다.
추천 결과에 나오더라도 "생산중단" 표시가 됩니다.

1. 당귀육황탕
2. 대청룡탕
3. 시호소간탕
4. 백출탕
5. 안태음
6. 보허탕
7. 인진호탕
8. 황금작약탕
9. 회춘양격산
10. 삼호작약탕

## 예시 시나리오

### 예시 1: 50세 여성, 갱년기

```
STEP 1: 여성, 성인, 비임신
STEP 2: 전신 증상 → general 계통 처방들 +10
STEP 3: 갱년기 증상 → 가미소요산 MFDS보너스 +8
STEP 4: 열 많음 → heat 일치 +5 / 기력 없음 → deficiency 일치 +4
STEP 5: 식은땀 +2, 입마름 +2
STEP 6: 만성 +3, 스트레스 악화 +4

→ 1위 가미소요산 (약 51점)
→ 2위 자음강화탕 (약 26점)
→ 3위 보중익기탕 (약 21점)
```

### 예시 2: 35세 남성, 감기 초기

```
STEP 1: 남성, 성인
STEP 2: 호흡기 → respiratory 처방들 +10
STEP 3: 갑자기 시작된 감기 → 오한+무한 → chills_no_sweat +8 +6
STEP 4: 추위 탐 → cold 일치 +5 / 기력 괜찮음 → excess 일치 +4
STEP 5: 손발 차다 +2
STEP 6: 급성 +3, 추위 악화 +4

→ 1위 갈근탕 (약 42점)
→ 2위 갈근해기탕 (약 38점)
→ 3위 구미강활탕 (약 28점)
```

### 예시 3: 40세 여성, 소화불량 + 스트레스

```
STEP 1: 여성, 성인, 비임신
STEP 2: 소화기 → digestive 처방들 +10
STEP 3: 속쓰림 → 반하사심탕 MFDS보너스 +6 / 스트레스성 → +5
STEP 4: 보통 체온 / 최근 악화 / 설사+변비 교대 → 반하사심탕 +4
STEP 5: 입마름 +2
STEP 6: 아급성 +3, 스트레스 악화 +4

→ 1위 반하사심탕 (약 44점)
→ 2위 반하후박탕 (약 22점)
→ 3위 평위산 (약 18점)
```
